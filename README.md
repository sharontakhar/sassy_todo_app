# Javascript sassy_todo_app 
To-do app which generates sassy functions when you've completed a task

*Contains functions such as-
*being able to add new todo task's below 
*Able to check off a task as Complete 
*Can delete a task when it is completed 
*If you check off a task you get a pop up with a sassy message as a pat on the back 
*Message pop up has been created using Toast.
